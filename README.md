<img src="./docs/preview.png" width="400">

## 使用

```shell
npx @shaoo/cssoo
```

## 注意

如果 cssoo 升级了版本，但使用 npx 命令安装的版本还是旧版本，需要先清除 npx 缓存

```shell
npx clear-npx-cache
```

## 已支持片段

- 优惠券
- 三角形
- 单行溢出省略号
- 多行溢出省略号
- 修改滚动条样式

## 待支持片段
